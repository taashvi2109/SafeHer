# SafeHer - Women Safety & Incident Reporting System

## Overview

SafeHer is a web-based women safety platform designed to help users report incidents, send SOS alerts, track complaint status, and provide feedback. The system includes separate dashboards for Users, Officers, and Admins to ensure efficient complaint management and response.

The project aims to provide a centralized platform for improving safety, communication, and incident tracking.

## Live Demo

https://safeher-dbzq.onrender.com/

## Features

### User Module

* User Registration & Login
* Report an Incident
* Submit SOS Alerts
* View Report Status
* Edit Profile
* Submit Feedback
* View Previous Reports

### Officer Module

* Officer Login
* View Assigned Reports
* View Report Details
* Update Report Status
* Manage Assigned Cases

### Admin Module

* Admin Login
* Manage Users
* Manage Officers
* View All Reports
* Assign Officers to Reports
* Monitor SOS Alerts
* View User Feedback
* Dashboard Statistics
